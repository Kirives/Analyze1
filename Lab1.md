#### Аффективные_расстройства, ПсихПоздВозр, НеврСтресс, УО, Эпилепсия, ОргПорЦНС, Химическая зависимость

Много пропусков - их можно принять за отсутствие данных.
Расстройства по лини матери передаются с большей вероятностью, поэтому их наличие считаем более значимым.
Если у критерия несколько значений через запятую, то каждому текстовому значению присваивается соответствующее числовое значение, а в итог вносится сумма числовых значений.

Закодируем признаки по значимости:

**Аффективные_расстройства**
- 0 - нет / пропуск / отец / дед по линии матери / бабка по линии отца
- 1 - бабка по линии матери / дядя/тётя по линии матери
- 2 - мать

**ПсихПоздВозр**
- 0 - нет / дед по линии матери / бабка по линии отца
- 1 - бабка по линии матери
- 2 - мать

**НеврСтресс**
- 0 - нет / отец
- 1 - бабка по линии матери / дядя/тётя по линии матери
- 2 - мать

**УО**
- Заполнена всего одна строка, можно удалить этот признак

**Эпилепсия**
- 0 - нет / отец / дядя/тётя по линии отца / дед по линии матери
- 1 - дядя/тётя по линии матери
- 2 - мать / сиблинги

**ОргПорЦНС**
- 0 - нет / отец
- 1 - бабка по линии матери / дядя/тётя по линии матери
- 2 - мать / сиблинги / дети

**Химическая зависимость**
- 0 - нет / отец / дед по линии матери / дед по линии отца / бабка по линии отца
- 1 - бабка по линии матери / дядя/тётя по линии матери
- 2 - мать / сиблинги

#### БерРодыМать

Для этого критерия также закодируем признаки по степени значимости.
Если у критерия несколько значений через запятую, то каждому текстовому значению присваивается соответствующее числовое значение, а в итог вносится сумма числовых значений.

- 0 – без патологии / не оценивались / пусто
- 1 – токсикоз первой половины беременности
- 2 – токсикоз второй половины беременности
- 3 – инфекционные заболевания / психотравмирующие события
- 4 – патология беременности / другие тяжёлые заболевания
- 5 – невынашивание беременности в анамнезе

#### ПатРодов, ПатРодовДруг

Будем рассматривать оба критерия вместе, сначала закодируем каждый из критериев отдельно, а затем объединим в один путём сложения.
Если у критерия несколько значений через запятую, то каждому текстовому значению присваивается соответствующее числовое значение, а в итог вносится сумма числовых значений.

**ПатРодов**
- 0 – нет / пропуск
- 1 – преждевременные роды / затяжные роды
- 2 – кесарево сечение
- 3 – гипоксия плода в родах
- 4 – наложение щипцов / стремительные роды / родился в асфиксии
- 5 – эклампсия (нефропатия + АД)

**ПатРодовДруг**
- 0 – нет / пропуск
- 3 - остальные, что-то равное гипоксии плода при родах в критерии ПатРодов

#### РанРазы

Этот критерий закодируем как распределение относительно нормального.

- -1 – наличие отклонений
- 0 – не оценивалось / пропуск
- 1 – в соответствии с возрастом

#### Ут2

В этом критерии очень много пропусков, а все заполненные данные уникальны.
Можно или оценивать по тяжести или бинарно есть / нет.
Для простоты можно удалить этот критерий.

#### НеротРасстрДетство

Для этого критерия также закодируем признаки по степени значимости.
Если у критерия несколько значений через запятую, то каждому текстовому значению присваивается соответствующее числовое значение, а в итог вносится сумма числовых значений.

- 1 – простые фобии / энурез / логоневроз
- 2 – симбиотическая привязанность к значимому взрослому / эмоциональная лабильность
- 3 – парейдолические переживания
- 4 – тяжёлые нарушения: ипохондрические / соматоформные / соматизированные расстройства / энкопрез / снохождение / сноговорение

#### НевротРасстрШкола, НевротРасстрШколаДоп

Будем рассматривать оба критерия вместе, сначала закодируем каждый из критериев отдельно, а затем объединим в один путём сложения.
Если у критерия несколько значений через запятую, то каждому текстовому значению присваивается соответствующее числовое значение, а в итог вносится сумма числовых значений.
